🎲 3D Cube Physics Simulation

Advanced Computational Methods Final Project

📊 About

A sophisticated 3D physics simulation demonstrating a cube bouncing inside a container with realistic force calculations using the Velocity Verlet numerical integration method.

Built with Python, NumPy, and Matplotlib for a computational methods course.

🔬 Physics Engine

Velocity Verlet Integration: More accurate than basic Euler method

Three Force Types: Gravity, Air Drag, Collision Forces

Energy Tracking: Kinetic, Potential, and Total Energy Conservation

📈 Visualization

Professional 3D trajectory rendering with real-time force magnitude plots. Advanced analysis tools provide detailed insights into collision events, terminal velocity, and energy dissipation.

✨ Key Features

  • Velocity Verlet Numerical Integration
  • Realistic Gravity Force (9.81 m/s²)
  • Air Drag Calculation (velocity squared)
  • Collision Detection & Response
  • 3D Trajectory Visualization
  • Real-time Force Analysis
  • Energy Conservation Tracking
  • Advanced Analysis Tools
  • Coefficient of Restitution Modeling
  • Detailed Statistics Output
